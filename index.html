PROJETS MARQUANTS

APPLICATION NUTRITION

Conception et développement d'une application mobile de suivi nutritionnel, publiée sur l'App Store. Permet aux utilisateurs d'organiser leurs repas, suivre leurs habitudes alimentaires et visualiser leur progression.

Projet : Application de nutrition

J'ai conçu une application de nutrition afin de trouver une solution pour améliorer la manière dont nous gérons notre alimentation, et comment nous nous organisons pour améliorer nos habitudes alimentaires, en rendant tout plus simple à gérer au quotidien.

L'objectif de l'application

Application mobile de suivi nutritionnel et d'accompagnement alimentaire, permettant aux utilisateurs de :

  • • Organiser leur alimentation
  • • Suivre leurs repas
  • • Visualiser leur progression
  • • Accéder à des outils d'aide à la décision

Architecture globale

UTILISATEUR (Mobile iOS / Android)
↓
Interface App (UI/UX)
↓
API Backend (serveur)
↓
Base de données
↓
Services externes (paiement, scan, etc.)

Fonctionnement détaillé

  1. 1/ Authentification utilisateur – Création de compte / connexion, gestion des profils, stockage sécurisé
  2. 2/ Gestion du profil – Objectifs personnels, préférences alimentaires, données de suivi
  3. 3/ Planification alimentaire – Génération ou organisation de repas, adaptation selon objectifs, structure journalière
  4. 4/ Suivi des repas – Ajout manuel des aliments, historique, visualisation des habitudes
  5. 5/ Scanner de produits – Scan via caméra, récupération des infos nutritionnelles, aide à la compréhension
  6. 6/ Suivi de progression – Visualisation des données dans le temps, feedback visuel
  7. 7/ Système d'abonnement – Accès fonctionnalités avancées, paiements intégrés via App Store

Gestion technique

  • • Backend centralisé (API)
  • • Communication sécurisée (HTTPS)
  • • Gestion des environnements (développement / production)
  • • Intégration des services mobiles (iOS / Android)

Stack fonctionnelle

  • • Frontend mobile (cross-platform)
  • • Backend API
  • • Base de données
  • • Services tiers (paiement, scan)

Dans le cadre d'un projet pour une entreprise spécialisée dans le bien-être, j'ai participé à la conception et au développement d'une application mobile dédiée au suivi alimentaire et à l'accompagnement des utilisateurs dans leurs habitudes nutritionnelles.

Sur le plan technique, l'application repose sur une architecture mobile connectée à un backend via API, permettant la gestion des données utilisateurs, des contenus et des interactions en temps réel.

Une attention particulière a été portée à l'expérience utilisateur, à la simplicité d'utilisation et à la structuration des fonctionnalités pour favoriser l'engagement et la régularité.

Ce projet m'a permis de développer des compétences en conception produit, structuration fonctionnelle, et intégration de services mobiles.

CRM ENTREPRISE

Développement d'un CRM métier sur mesure pour une entreprise souhaitant structurer sa gestion commerciale et optimiser le suivi de ses prospects.

Projet : Développement d'un CRM métier

J'ai conçu et développé un CRM (Customer Relationship Management) destiné à une entreprise souhaitant structurer et optimiser sa gestion commerciale et relation client.

Objectif du CRM

Le système permet de :

  • • Centraliser les informations clients et prospects
  • • Suivre les interactions commerciales
  • • Organiser les tâches et les relances
  • • Planifier des événements (rendez-vous, appels, suivis)
  • • Améliorer la conversion des prospects en clients

Architecture technique

  • Frontend – Interface utilisateur permettant visualisation des prospects, suivi des actions, gestion quotidienne
  • Backend (API) – API REST gérant authentification, création/modification prospects, organisation tâches, planification événements
  • Base de données – Base de données NoSQL stockant utilisateurs, prospects, tâches, événements

Fonctionnement global

  1. 1. Un utilisateur se connecte à la plateforme
  2. 2. Il accède à un tableau de bord centralisé
  3. 3. Il peut ajouter un prospect, suivre son évolution, planifier une action
  4. 4. Les données sont stockées et synchronisées en temps réel
  5. 5. Le système permet un suivi structuré du cycle de vente

Valeur apportée

Ce CRM permet à une entreprise de :

  • • Structurer son pipeline commercial
  • • Gagner du temps dans la gestion quotidienne
  • • Améliorer le suivi des prospects
  • • Augmenter le taux de conversion

Technologies utilisées

  • • Node.js (backend)
  • • Express (API)
  • • MongoDB (base de données)
  • • JavaScript
